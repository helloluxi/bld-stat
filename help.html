BLD Stat Tools - Help

Probability

The Probability tab allows you to calculate the probability of BLD cycle configurations in a random state.

How algs are counted?

  • Normal 3-style: 2 letter as one alg.
  • Parity: 0.5 algs for edge and 0.5 algs for corner.
  • Float 3-cycles are counted as 1 alg instead of 2 algs (even though it is 4 letters).
  • Edge 2-flip or 4-flip, corner 2-twist or 3-twist are counted as 1 alg.

Distribution Tab

The Distribution tab provides visual analysis of statistical distributions for both edges and corners.

Custom Expression Input

You can enter custom JavaScript expressions to analyze different aspects of the data. The default expression is algs.

Available Variables

When writing custom expressions, you can use these variables:

  • parity - Parity value (0 for even, 1 for odd)
  • breaks - Number of cycle breaks
  • algs - Algorithm count/length
  • float1 through float5 - Floating n-cycle
  • bad1 through bad5 - Non-floating n-cycle (e.g., bad1 for flips/twists)
